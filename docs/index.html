Term of the Day

Sharpe Ratio

A risk-adjusted performance metric calculated as (Portfolio Return − Risk-Free Rate) ÷ Standard Deviation. A ratio above 1 is considered good; above 2 is excellent. It rewards high returns while penalising high volatility, making it the standard tool for comparing strategies that carry different risk profiles. The Sortino Ratio is a variant that penalises only downside volatility.
